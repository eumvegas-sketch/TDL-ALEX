# TO-DO-LIST

โปรเจกต์นี้คือ To-do list ส่วนตัวของฉัน ใช้สำหรับบันทึกงาน เป้าหมาย ความคืบหน้า และไอเดียต่าง ๆ ผ่าน GitHub

## วิธีใช้งานแบบง่าย

1. เปิดไฟล์ `TODO.md`
2. เพิ่มรายการงานใหม่ด้วยรูปแบบ `- [ ] ชื่องาน`
3. ถ้าทำเสร็จแล้ว เปลี่ยนเป็น `- [x] ชื่องาน`
4. กด Commit เพื่อบันทึกความคืบหน้า

## โครงสร้างไฟล์

```text
TO-DO-LIST/
├── README.md
├── TODO.md
├── projects/
│   ├── personal.md
│   ├── work.md
│   └── ideas.md
└── progress/
    └── 2026-05.md
```

## ตัวอย่าง Commit

- `เพิ่มรายการงานประจำวัน`
- `อัปเดตงานที่ทำเสร็จแล้ว`
- `เพิ่มไอเดียโปรเจกต์ใหม่`
- `สรุปความคืบหน้าประจำเดือน`
